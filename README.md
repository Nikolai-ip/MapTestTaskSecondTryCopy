## Implemented Features
1. Add a new pin by clicking on the map.
2. Move a pin by long pressing on it.
3. Edit pin information.
4. Upload images to the project by copying them to the persistent folder.
5. Save pins and data by clicking the “Save” button.
6. Edit pin information. - Delete a pin.
7. Window and pin animations.
